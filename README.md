# 🧠 Outfit Intelligence Tool

A client-side outfit analysis web application that uses real color theory to score and analyze your outfits.

## Features
- Multi-input system (manual color selection, camera capture, image upload)
- Auto garment zone detection (Top, Bottom, Shoes)
- Strict color harmony analysis
- Psychological confidence scoring
- Mood detection
- Outfit comparison mode
- Professional responsive dark-themed UI
- 100% client-side — no server required
